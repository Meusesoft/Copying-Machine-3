Menu file

New...

This command will scan a page from the scanner and create a new document.

Open...

This command will open a previously saved document from a drive.

Save

This command will save the currently selected document to the drive.

Save as...

This command will save the currently selected document to the drive. A dialog will be opened to select the destination and the file name.

Close...

This command will close the currently selected document.

Print...

This command will print the currently selected document.

Print preview...

This command will switch the view between 'print preview' and 'normal view'. In print preview mode a page will be shown with a preview of how the document will be printed upon it. 

Preferences...

This command will open the preferences dialog. In this dialog a lot of preferences can be set. The preferences are seperated in groups, for example General, Scanner and Printer.

Acquire...

This command will scan a page from the scanner and create a new document.

Select source...

This command will open a dialog in which all available scanning devices are enumerated. A default scanning device can be selected.

Exit...

This command will close Copying Machine.
