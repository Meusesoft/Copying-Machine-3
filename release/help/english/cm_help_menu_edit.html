Menu edit

Cut

This command will cut the selection to the clipboard. The selection will be filled with the current color.

Copy

This command will copy the selection to the clipboard.

Paste as new document

This command will paste the contents of the clipbard as a new document.

Paste as new page

This command will paste the contents of the clipboard as a new page in the current document.

Select all

This command will select the complete current page.

Unselect

This command will remove the selection from the current page.

Invert selection

This command will switch the selection mode between inside and outside. The selection mode is used in the ´fill selection´ tool. In the inside mode the inside of the selection will be filled and in the outside mode the outside.
